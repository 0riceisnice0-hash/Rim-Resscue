# Rim Rescue

Professional alloy wheel refurbishment and repair services based in the UK.

## About

Rim Rescue specialises in alloy wheel refurbishment, diamond cut repair, kerb damage repair, and custom finishes. Operating since 2020, we take pride in delivering high-quality results for every wheel we restore.
